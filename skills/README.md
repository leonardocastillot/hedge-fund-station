# Hedge Fund Skills

This folder contains workspace-specific skills for agents operating in Hedge Fund Station.

Each subfolder should contain a `SKILL.md` file with:

- when to use the skill
- required inputs
- expected outputs
- repo paths to inspect first
- workflow steps
- validation rules

- `hedge-fund-strategy-lab`
- `hedge-fund-strategy-review`
