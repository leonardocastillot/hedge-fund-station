# Media Artifact Archive

The source repo should not carry rendered marketing videos or frame captures.
They are delivery artifacts, not application source, strategy evidence, or
harness state.

The following packages are local-only or external-archive material:

- `videos/hedge-fund-station-epic/`
- `renders/hedge-fund-station-epic.mp4`
- `renders/hedge-fund-station-epic-capcut.mp4`
- `renders/hedge-fund-station-epic-capcut-mobile.mp4`

Keep reusable product code, backend strategy logic, docs, tests, and stable
commands in git. Keep rendered media in local storage, release storage, or an
external asset archive and reference it from handoffs when needed.
